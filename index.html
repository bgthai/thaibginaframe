บทที่ หนึ่ง : สำรวจกองทัพที่สมรภูมิ กุรุกฺเษตฺร

ขณะที่กองทัพทั้งสองฝ่ายเตรียมพร้อมที่จะรบ อรฺชุนนักรบผู้กล้าหาญทรงเห็นญาติสนิทมิตรสหาย และครูบาอาจารย์ที่อยู่ในกองทัพทั้งสองฝ่ายพร้อมที่จะรบ และสังเวยชีวิตของตนเอง อรฺชุนทรงถูกความเศร้าสลด และความสงสารครอบงํา รู้สึกหมดกําลังใจในการรบ รู้สึกสับสน และคิดจะล้มเลิกความตั้งใจที่จะรบ

บทที่ สอง : บทสรุป ภควัท-คีตา

 อรฺชุน ทรงยอมมาเป็นสาวกขององค์กฺฤษฺณ องค์กฺฤษฺณจึงทรงเริ่มสอน อรฺชุนด้วยการอธิบายพื้นฐานของข้อแตกต่างระหว่างร่างกายวัตถุอันไม่ถาวร และดวงวิญญาณที่เป็นอมตะ องค์กฺฤษฺณทรงอธิบายถึงระบบการเปลี่ยนร่างกายธรรม ชาติแห่งการรับใช้องค์ภควานฺโดยไม่เห็นแก่ตัว และบุคลิกลักษณะของบุคคลผู้รู้แจ้งตนเอง

บทที่ สาม : กรฺม-โยค

ทุกคนต้องมีกิจกรรมบางอย่างในโลกวัตถุนี้แต่กิจกรรมนั้นอาจพันธนาการตัวเราให้อยู่ในโลกวัตถุนี้ หรือมันจะทําให้เราหลุดพ้นไปจากโลกนี้ได้ ด้วยการปฎิบัติกิจกรรมที่ทําให้องค์ภควานฺทรงพอพระทัยโดยปราศจากความเห็นแก่ตัวนั้นจะสามารถทําให้เราหลุดพ้นจากกฎแห่งกรรมและบรรลุถึงความรู้ทิพย์แห่งตนเองและองค์ภควานฺ

บทที่ สี่ : ความรู้ทิพย์

ความรู้ทิพย์ คือ ความรู้แห่งดวงวิญญาณ แห่งองค์ภควานฺ และความสัมพันธ์ระหว่างดวงวิญญาณและองค์ภควานฺเป็นความรู้ที่ให้ทั้งความบริสุทธิ์และความหลุดพ้น ความรู้เช่นนี้เป็นผลมาจากการปฎิบัติอุทิศตนเสียสละรับใช้โดยปราศจากความเห็นแก่ตัว (กรฺม-โยค) พระองค์ทรงอธิบายถึงประวัติศาสตร์โบราณของ คีตา จุดมุ่งหมายและความสําคัญที่พระองค์ทรงเสด็จลงมาในโลกวัตถุนี้เป็นครั้งคราว และความจําเป็นในการเข้าพบ กุรุ หรือครูผู้รู้แจ้งตนเอง

บทที่ ห้า : กรฺม-โยค การปฏิบัติในกฺฤษฺณจิตสำนึก

จากภายนอกเขาอาจปฎิบัติงานทุกชนิดแต่ภายในสละผลของงานทั้งหมด ดังนั้นผู้มีปัญญาที่บริสุทธิ์จากไฟแห่งความรู้ทิพย์จะบรรลุถึงความสงบ ไม่ยึดติด มีความอดทน มีจักษุทิพย์ และมีความปลื้มปีติสุข

บทที่ หก : ธฺยาน-โยค

อษฺฏางฺค-โยค การฝึกสมาธิคล้ายเครื่องจักรกล ควบคุมจิตใจและประสาทสัมผัสตั้งสมาธิอยู่ที่ ปรมาตฺมา (องค์อภิวิญญาณผู้ทรงประทับอยู่ภายในหัวใจของมวลชีวิต) การปฎิบัติเช่นนี้จะนําไปสู่สมาธิมีจิตสํานึกอยู่ที่องค์ภควานฺอย่างสมบูรณ์

บทที่ เจ็ด : ความรู้แห่งสัจธรรม

องค์กฺฤษฺณทรงเป็นสัจธรรมสูงสุด ทรงเป็นแหล่งกําเนิดและพลังค้ําจุนสูงสุดของสรรพสิ่งทั้งวัตถุและทิพย์ ดวงวิญญาณที่เจริญแล้วจะศิโรราบต่อพระองค์ด้วยการอุทิศตนเสียสละ ขณะที่ดวงวิญญาณผู้ไม่มีบุญจะหันเหจิตใจของตนเองไปบูชาพระปฎิมาองค์อื่น

บทที่ แปด : การบรรลุถึงองค์ภควานฺ

จากการระลึกถึงองค์กฺฤษฺณ ด้วยการอุทิศตนเสียสละตลอดชีวิตของเราโดยเฉพาะในขณะที่เรากําลังจะตาย เราจะสามารถบรรลุถึงพระตําหนักสูงสุดของพระองค์ซึ่งอยู่หนือโลกวัตถุนี้

บทที่ เก้า : ความรู้ที่ลับสุดยอด

องค์กฺฤษฺณทรงเป็นองค์ภควานฺและทรงเป็นพระปฎิมาสูงสุดแห่งการบูชา ดวงวิญญาณมีความสัมพันธ์กับพระองค์นิรันดรโดยผ่านทาง ภกฺติ-โยค หรือการอุทิศตนเสียสละรับใช้ทิพย์ จากการฟื้นฟูการอุทิศตนเสียสละที่บริสุทธิ์เราจะสามารถกลับคืนสู่องค์กฺฤษฺณในอาณาจักรทิพย์

บทที่ สิบ : ความมั่งคั่งแห่งสัจธรรม

ปรากฏการณ์อันน่าอัศจรรย์ทั้งหลายที่แสดงถึงพลังอํานาจ ความสุนทรีย์ ความยิ่งใหญ่ หรือความประเสริฐนั้น ไม่ว่าในโลกวัตถุหรือในโลกทิพย์เป็นเพียงส่วนหนึ่งแห่งการปรากฏของพลังงานทิพย์ และความมั่งคั่งขององค์กฺฤษฺณ ในฐานะที่ทรงเป็นแหล่งกําเนิดสูงสุดของแหล่งกําเนิดทั้งปวงองค์กฺฤษฺณ ทรงเป็นผู้ค้ําจุนและทรงเป็นแก่นสารของทุกสิ่งทุกอย่าง พระองค์ทรงเป็นพระปฎิมาสูงสุดแห่งการบูชาสําหรับมวลชีวิต

บทที่ สิบเอ็ด : รูปลักษณ์จักรวาล

องค์กฺฤษฺณทรงประทานจักษุทิพย์แด่ อรฺชุน และทรงเปิดเผยรูปลักษณ์จักรวาลอันอลังการที่ไร้ขอบเขต เช่นนี้ทรงสถาปนาความเป็นทิพย์โดยสรุปองค์กฺฤษฺณทรงอธิบายว่ารูปลักษณ์คล้ายมนุษย์ที่มีความสง่างามสูงสุดเป็นรูปลักษณ์เดิมแท้ขององค์ภควานฺ เราสามารถสําเหนียกรูปลักษณ์นี้ด้วยการอุทิศตนเสียสละรับใช้ที่บริสุทธิ์เท่านั้น

บทที่ สิบสอง : การอุทิศตนเสียสละรับใช้

ภกฺติ-โยค หรือการอุทิศตนเสียสละรับใช้ที่บริสุทธิ์ต่อองค์กฺฤษฺณนั้นเป็นวิธีที่สูง สุด และเหมาะสมที่สุดในการบรรลุถึงความรักอันบริสุทธิ์ต่อพระองค์ ซึ่งเป็นจุดมุ่งหมายสูงสุดแห่งความเป็นอยู่ทิพย์ พวกที่ปฎิบัติตามวิถีทางที่สูงสุดนี้จะพัฒนาคุณสมบัติทิพย์

บทที่ สิบสาม : ธรรมชาติ ผู้รื่นเริง และจิตสำนึก

ผู้ที่เข้าใจข้อแตกต่างระหว่างร่างกาย ปัจเจกวิญญาณ และอภิวิญญาณ ผู้ที่อยู่เหนือทั้งร่างกาย และปัจเจกวิญญาณจะบรรลุถึงความหลุดพ้นจากโลกวัตถุนี้

บทที่ สิบสี่ : สามระดับแห่งธรรมชาติวัตถุ

ดวงวิญญาณที่อยู่ในร่างกายทั้งหมดอยู่ภายใต้การควบคุมของสามระดับ หรือสามลักษณะของธรรมชาติวัตถุ คือ ความดี ตัณหา และอวิชชา องค์กฺฤษฺณทรงอธิบายว่าธรรมชาติวัตถุคืออะไร มีผลกระทบต่อเราอย่างไร เราจะสามารถข้ามพ้นระดับแห่งธรรมชาติวัตถุได้อย่างไร และลักษณะอาการของผู้ที่บรรลุถึงระดับทิพย์เป็นอย่างไร

บทที่ สิบห้า : โยคะแห่งองค์ภควานฺ

จุดมุ่งหมายสูงสุดของความรู้พระเวท คือ การทําให้ตนเองหลุดพ้นจากพันธนาการแห่งโลกวัตถุ และเข้าใจว่าองค์กฺฤษฺณทรงเป็นบุคลิกภาพสูงสุดแห่งพระเจ้า ผู้ที่เข้าใจบุคลิกภาพสูงสุดแห่งองค์กฺฤษฺณจะศิโรราบและปฎิบัติการอุทิศตนเสียสละรับใช้ต่อพระองค์

บทที่ สิบหก : ธรรมชาติทิพย์และธรรมชาติมาร

พวกที่มีคุณสมบัติมารและใช้ชีวิตตามอําเภอใจโดยไม่ปฎิบัติตามหลักคําสอนของพระคัมภีร์จะไปเกิดในที่ต่ําและถูกพันธนาการทางวัตถุมากยิ่งขึ้น แต่พวกที่มีคุณสมบัติทิพย์ใช้ชีวิตอย่างมีวินัยและปฏิบัติตามคําสอนของพระคัมภีร์จะค่อยๆบรรลุถึงความสมบูรณ์ทิพย์

บทที่ สิบเจ็ด : ระดับแห่งความศรัทธา

มีความศรัทธาอยู่สามระดับซึ่งสัมพันธ์กับสามระดับแห่งธรรมชาติวัตถุ การปฏิบัติของพวกที่มีความศรัทธาอยู่ในระดับตัณหา และอวิชชาจะได้รับผลทางวัตถุที่ไม่ถาวรเท่านั้น ขณะที่การปฏิบัติที่อยู่ในระดับความดีตามคําสอนของพระคัมภีร์จะทําให้หัวใจบริสุทธิ์นําไปสู่ความศรัทธาที่บริสุทธิ์ และการอุทิศตนเสียสละรับใช้ที่บริสุทธิ์ต่อองค์กฺฤษฺณ

บทที่ สิบแปด : บทสรุปความสมบูรณ์แห่งการเสียสละ

องค์กฺฤษฺณ ทรงอธิบายถึงความหมายแห่งการเสียสละและผลของสามระดับแห่งธรรมชาติวัตถุที่มีต่อจิตสํานึกและกิจกรรมของมนุษย์ พระองค์ทรงอธิบายถึงความรู้แจ้ง บระฮมัน พระบารมีของ ภควัท-คีตา และข้อสรุปสุดท้ายของ คีตา คือ วิถีทางสูงสุดของศาสนาอันเป็นสัจธรรม นั่นก็คือการศิโรราบด้วยความรักต่อองค์กฺฤษฺณ อย่างไร้ข้อแม้ซึ่งจะทําให้เราเป็นอิสระจากความบาปทั้งปวงจะนําเรามาสู่ความรู้แจ้งที่สมบูรณ์ และทําให้เรากลับคืนสู่พระตําหนักทิพย์นิรันดรขององค์ศฺรีกฺฤษฺณ
